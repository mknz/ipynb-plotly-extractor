# ipynb-plotly-extractor

CLI utility to extract Plotly JSON data from IPython Notebook file

## Quickstart

```bash
git clone https://github.com/mknz/ipynb-plotly-extractor.git
cd ./ipynb-plotly-extractor
sudo npm install -g
```

```bash
ipex Untitled.ipynb
ls Untitled*.json
>>> Untitled_fig_001.json
```
